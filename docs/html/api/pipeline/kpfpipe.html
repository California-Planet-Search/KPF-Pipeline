KPF Pipeline¶

class kpfpipe.pipelines.kpfpipeline.KPFPipeline(context: keckdrpframework.models.processing_context.ProcessingContext)[source]¶

Pipeline to Process KPF data using the KeckDRPFramework

Parameters

context (ProcessingContext) – context class provided by the framework

event_table¶

table of actions known to framework. All primitives must be registered here.

Type

dictionary

Note

The correct operation of the recipe visitor depends on action.args being KpfArguments, which is an extension (class derived from) the Keck DRPF Arguments class. All pipeline primitives must use KpfArguments rather than simply Arguments for their return values. They will also get input arguments packaged as KpfArguments.
