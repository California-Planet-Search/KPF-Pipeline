valid_level1_data()[source]¶

Confirms that KPF1 object has necessary data/structure to operate on (after checking for its existence) The first checks are similar to valid_level0_data(), but we keep them separated because different checks may be necessary e.g., checking that we only have 1-d spectra for each chip now, the correct number of orderlets, etc.

bool
